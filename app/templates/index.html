Number of Rounds
The number of rounds indicates how many times each strategy will play against its opponent in a single game. Increasing the number of rounds can provide more accurate results by averaging out luck or chance elements.
Number of Games
The total number of games determines how many distinct matchups will occur in the competition. Each game will involve different strategies or different configurations of the same strategies.
Points Per Game
The points per game represent the scoring mechanism for the competition. Each game will award points to the winning strategy, which accumulates to determine overall performance.
